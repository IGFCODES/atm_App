 What the Mini ATM App featur will do:
1. Greet the user
2. Show their balance
3. Allow them to deposite and withdraw money.
4. Ask the user who they want to send money to (receiver's name or account number).
5. Ask how much they want to transfer.
6. Check if they have enough money.
7. Deduct the amount and show a success message
8. Keep track of every transaction (Deposit, Withdraw, Transfer).
9. Store the details (like type, amount, and date).
10. Let the user view all past transactions anytime.
End with a thank you message
